# IndexOptions: AI-Powered Gamma & Flow Analytics
A high-performance trading dashboard that transforms complex SPX options data into actionable visual intelligence.

![Alt text](https://github.com/dearvn/indexoptions/raw/main/screenshot-spx.png?raw=true "SPX")


Visual Gamma Mapping: Converts dense GEX tables into an intuitive horizontal heatmap to instantly identify institutional "Liquidity Walls."

Actionable Intelligence: A Smart Recommendation engine that provides real-time trade setups (Entry/Target/Stop) based on market friction points.

Tiered Alerting: High-priority signal filtering that separates critical "Wall Breaks" from market noise.

Dynamic Sentiment: Features the S-GEX Dial and Gamma Flip Visualizer for a 1-second assessment of market bias.

Tech Stack: React 18, TypeScript, Tailwind CSS, Replit AI.
